# 📱 دليل نشر تطبيق خافِت على App Store

هذا المجلد (`khafet-app`) جاهز — يلفّ موقع خافِت داخل تطبيق آيفون عبر **Capacitor**.

---

## المرحلة ١: جهّزي الماك السحابي
1. اشتركي في **MacinCloud** (أو أي ماك سحابي) — اختاري خطة فيها **Xcode مثبّت**.
2. سجّلي دخول للماك عن بُعد (Remote Desktop).
3. تأكدي إن هذي مثبّتة (غالباً موجودة):
   - **Xcode** (من Mac App Store)
   - **Node.js** — لو مو موجود: نزّليه من nodejs.org
   - **CocoaPods** — لو مو موجود، في Terminal: `sudo gem install cocoapods`

## المرحلة ٢: ارفعي المجلد
- ارفعي مجلد `khafet-app` كامل للماك السحابي (سطح المكتب مثلاً).

## المرحلة ٣: الأوامر (في Terminal)
افتحي **Terminal**، وروحي لمجلد التطبيق ثم شغّلي:

```bash
cd ~/Desktop/khafet-app        # عدّلي المسار لمكان المجلد عندك

npm install                    # يركّب Capacitor
npx cap add ios                # ينشئ مشروع iOS
npx cap sync                   # ينسخ الموقع + الإضافات
npx cap open ios               # يفتح المشروع في Xcode
```

## المرحلة ٤: إعدادات Xcode
بعد ما يفتح Xcode، اضغطي على **App** (فوق يسار) ثم:

1. **Signing & Capabilities**:
   - ✅ فعّلي «Automatically manage signing».
   - **Team**: اختاري حساب المطوّر (fadi Alzahrani).
   - **Bundle Identifier**: `com.khafet.app` (أو غيّريه لأي شي فريد).

2. **General → Display Name**: `خافِت`.

3. **الأيقونة**: `App → Assets → AppIcon` — اسحبي صورة اللوقو (١٠٢٤×١٠٢٤).

4. **أذونات (مهم — بدونها الموقع/الكاميرا ما تشتغل)**:
   افتحي `Info.plist` وأضيفي هذي المفاتيح:
   | المفتاح | القيمة |
   |---|---|
   | `NSLocationWhenInUseUsageDescription` | نحتاج موقعك لتحديد مكان الخدمة على الخريطة |
   | `NSCameraUsageDescription` | لتصوير المشكلة أو المنزل |
   | `NSPhotoLibraryUsageDescription` | لإرفاق صورة من المعرض |

## المرحلة ٥: جرّبي
- اختاري محاكي آيفون فوق، واضغطي **▶️ Run** — يفتح التطبيق. جرّبي الحجز والخريطة.

## المرحلة ٦: الرفع للـ App Store
1. فوق: بدّلي الجهاز إلى **Any iOS Device**.
2. القائمة: **Product → Archive** — ينتظر شوي.
3. لما يخلص: **Distribute App → App Store Connect → Upload**.

## المرحلة ٧: App Store Connect
1. افتحي **appstoreconnect.apple.com → Apps → +** → أنشئي التطبيق (نفس Bundle ID).
2. عبّي: الاسم، الوصف، التصنيف، **لقطات شاشة**، و**رابط سياسة الخصوصية** (إلزامي).
3. اختاري البناء اللي رفعتيه، ثم **Submit for Review**.
4. المراجعة ~١-٣ أيام.

---

## 🔄 تحديث التطبيق لاحقاً
لما تعدّلين الموقع:
1. انسخي ملفات `khafet-site` الجديدة فوق `khafet-app/www/`.
2. `npx cap sync` ثم `Archive` وارفعي نسخة جديدة (ارفعي رقم الإصدار في Xcode).

## ⚠️ ملاحظات مهمة
- 🔔 **الإشعارات الفورية**: أبل قد ترفض «موقع ملفوف» بلا مميزات أصلية. أقوى إضافة = Push Notifications (نضيفها كخطوة تالية — تحتاج إعداد إضافي).
- 📍 **خريطة الموقع (GPS)**: تشتغل مع إضافة `@capacitor/geolocation` المضمّنة + أذونات Info.plist.
- 🔒 **سياسة الخصوصية**: لازم رابط لها في App Store Connect (نجهّزها لك بسهولة).
- كل CDN (Supabase/الخرائط/الخطوط) على HTTPS — فما تحتاجين إعدادات ATS إضافية.
